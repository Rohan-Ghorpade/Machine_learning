| Algorithm                          | Problem Type                | Best Used On What Kind of Data                  | When to Use                                         | Advantages                             | Disadvantages                        |
| ---------------------------------- | --------------------------- | ----------------------------------------------- | --------------------------------------------------- | -------------------------------------- | ------------------------------------ |
| Linear Regression                  | Regression                  | Numerical tabular data with linear relationship | Predict continuous values like salary, sales, price | Simple, fast, interpretable            | Cannot capture complex patterns      |
| Logistic Regression                | Classification              | Binary classification datasets                  | Fraud detection, spam detection                     | Fast, explainable, lightweight         | Poor on nonlinear data               |
| Decision Tree                      | Classification & Regression | Small-medium structured data                    | Rule-based predictions                              | Easy visualization, interpretable      | Overfitting problem                  |
| Random Forest                      | Classification & Regression | Tabular data with mixed features                | General-purpose baseline model                      | High accuracy, handles missing values  | Large model size, slower inference   |
| XGBoost                            | Classification & Regression | Large tabular datasets                          | Banking, fraud, churn prediction                    | Very high accuracy, feature importance | Longer training time                 |
| LightGBM                           | Classification & Regression | Huge datasets                                   | Fast enterprise systems                             | Very fast, low memory                  | Sensitive to noise                   |
| CatBoost                           | Classification & Regression | Categorical-heavy datasets                      | Banking/customer data                               | Handles categorical data directly      | Training can be slower               |
| Support Vector Machine (SVM)       | Classification              | Medium-sized numerical datasets                 | Text classification, medical data                   | Effective in high dimensions           | Slow for large datasets              |
| K-Nearest Neighbors (KNN)          | Classification & Regression | Small datasets                                  | Similarity-based problems                           | Simple, no training required           | Very slow prediction                 |
| Naive Bayes                        | Classification              | Text/NLP datasets                               | Spam filtering, sentiment analysis                  | Very fast, works well on text          | Assumes feature independence         |
| Gradient Boosting                  | Classification & Regression | Structured/tabular datasets                     | High-accuracy enterprise ML                         | Strong performance                     | Slow training                        |
| AdaBoost                           | Classification              | Clean datasets with weak learners               | Improving weak models                               | Simple boosting technique              | Sensitive to noisy data              |
| Extra Trees Classifier             | Classification              | Large tabular datasets                          | Fast alternative to Random Forest                   | Faster training                        | Less interpretable                   |
| Ridge Regression                   | Regression                  | Multicollinearity datasets                      | Reduce overfitting in linear regression             | Stable coefficients                    | Still linear assumption              |
| Lasso Regression                   | Regression                  | High-dimensional data                           | Feature selection problems                          | Removes irrelevant features            | Can underfit                         |
| ElasticNet                         | Regression                  | Highly correlated features                      | Mixed Ridge + Lasso benefits                        | Balanced regularization                | Hyperparameter tuning required       |
| Neural Networks (MLP)              | Classification & Regression | Large complex datasets                          | Nonlinear patterns                                  | Captures complex relationships         | Needs more data and tuning           |
| CNN (Convolutional Neural Network) | Classification              | Image/video datasets                            | Face detection, document verification               | Excellent image understanding          | High compute required                |
| RNN / LSTM                         | Sequence Prediction         | Sequential/time-series/text data                | Forecasting, NLP                                    | Handles sequences well                 | Slow training                        |
| Transformer Models                 | NLP & Sequence Tasks        | Large text datasets                             | Chatbots, LLMs, translation                         | State-of-the-art NLP                   | Very resource intensive              |
| Isolation Forest                   | Anomaly Detection           | High-dimensional tabular data                   | Fraud/anomaly detection                             | Good for rare anomalies                | Not ideal for labeled classification |
| One-Class SVM                      | Anomaly Detection           | Small anomaly datasets                          | Outlier detection                                   | Works with unlabeled anomalies         | Poor scalability                     |
| Autoencoders                       | Anomaly Detection           | Complex/high-dimensional data                   | Fraud, image anomaly detection                      | Learns hidden patterns                 | Requires deep learning expertise     |
